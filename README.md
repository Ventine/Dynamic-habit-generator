## ✨ Características Principales

- **🎲 Generación Aleatoria Inteligente:** Crea un horario diario único basado en un pool de tareas semanal.
- **🧠 Restricción Algorítmica:** Garantiza tiempos de descanso equilibrados (mínimo 1, máximo 2 por día).
- **💾 Persistencia Local:** Todo el estado, progreso e historial se guarda de forma segura en el `localStorage` del navegador. Ninguna base de datos externa requerida.
- **🏆 Gamificación:** Historial de victorias, animaciones de recompensa (confetti) y UI reactiva.
- **⚡ Inyección Dinámica:** Capacidad de agregar tareas "Extra" en tiempo real sin alterar el pool semanal.
